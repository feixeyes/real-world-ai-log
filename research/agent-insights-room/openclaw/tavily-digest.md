## Query (general): openclaw official documentation

## Answer

OpenClaw is an open-source framework for running AI agents. It connects messaging platforms to AI coding agents. It can be installed via npm and runs on any operating system.

## Query (general): openclaw docs API reference

## Answer

OpenClaw is a local AI assistant that connects messaging services to coding agents. It uses Ollama for integration and requires a context window of at least 64k tokens. The API is open-source and supports multi-step task planning.
